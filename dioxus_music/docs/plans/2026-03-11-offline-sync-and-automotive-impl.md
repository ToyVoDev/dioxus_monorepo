# Offline Sync & Automotive UI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add seamless offline playback with automatic connectivity detection, local caching/sync on desktop and mobile, and Android Auto + CarPlay support.

**Architecture:** All clients (desktop, mobile, web) connect to a remote server (Postgres). Desktop uses SQLite (Diesel) for local cache; Android uses Room; iOS uses Core Data. A `PlatformAudio` / `PlatformStorage` trait abstraction bridges Rust and native code. Android Auto and CarPlay are native platform services that share the local database and playback engine with the main app.

**Tech Stack:** Rust (Dioxus 0.7, Diesel 2 with SQLite feature), Kotlin (Room, ExoPlayer, MediaBrowserServiceCompat), Swift (Core Data, AVQueuePlayer, CPTemplate), UniFFI for Rust-Kotlin/Swift bridging.

**Design doc:** `docs/plans/2026-03-11-offline-sync-and-automotive-design.md`

```rust
pub async fn stream_track(
    Extension(pool): Extension<DbPool>,
    Path(track_id): Path<Uuid>,
) -> Result<Response, (StatusCode, &'static str)> {
    let mut conn = pool
        .get()
        .await
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "DB pool error"))?;

    let file_path: String = tracks::table
        .filter(tracks::id.eq(track_id))
        .select(tracks::file_path)
        .first(&mut conn)
        .await
        .map_err(|_| (StatusCode::NOT_FOUND, "Track not found"))?;

    let path = PathBuf::from(&file_path);

    let ext = path
        .extension()
        .and_then(|e| e.to_str())
        .unwrap_or("")
        .to_lowercase();

    let content_type = match ext.as_str() {
        "flac" => "audio/flac",
        "mp3" => "audio/mpeg",
        "ogg" | "opus" => "audio/ogg",
        _ => "application/octet-stream",
    };

    let metadata = tokio::fs::metadata(&path)
        .await
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "Failed to read file metadata"))?;

    let file = tokio::fs::File::open(&path)
        .await
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "Failed to open file"))?;

    let stream = ReaderStream::new(file);
    let body = Body::from_stream(stream);

    Ok(Response::builder()
        .header(header::CONTENT_TYPE, content_type)
        .header(header::CONTENT_LENGTH, metadata.len())
        .body(body)
        .unwrap())
}
```

```rust
pub async fn cover_art(
    Extension(pool): Extension<DbPool>,
    Path(track_id): Path<Uuid>,
) -> Result<Response, (StatusCode, &'static str)> {
    let mut conn = pool
        .get()
        .await
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "DB pool error"))?;

    let file_path: String = tracks::table
        .filter(tracks::id.eq(track_id))
        .select(tracks::file_path)
        .first(&mut conn)
        .await
        .map_err(|_| (StatusCode::NOT_FOUND, "Track not found"))?;

    let path_clone = file_path.clone();
    let tagged_file = tokio::task::spawn_blocking(move || lofty::read_from_path(&path_clone))
        .await
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "Task join error"))?
        .map_err(|_| (StatusCode::INTERNAL_SERVER_ERROR, "Failed to read tags"))?;

    let picture = tagged_file
        .primary_tag()
        .or_else(|| tagged_file.first_tag())
        .and_then(|tag| {
            tag.pictures()
                .iter()
                .find(|p| p.pic_type() == lofty::picture::PictureType::CoverFront)
                .or_else(|| tag.pictures().first())
        })
        .ok_or((StatusCode::NOT_FOUND, "No cover art found"))?;

    let mime = picture.mime_type().unwrap_or(&lofty::picture::MimeType::Jpeg);
    let content_type = match mime {
        lofty::picture::MimeType::Png => "image/png",
        lofty::picture::MimeType::Jpeg => "image/jpeg",
        lofty::picture::MimeType::Bmp => "image/bmp",
        _ => "image/jpeg",
    };

    let data = picture.data().to_vec();

    Ok(Response::builder()
        .header(header::CONTENT_TYPE, content_type)
        .header(header::CONTENT_LENGTH, data.len())
        .header(header::CACHE_CONTROL, "public, max-age=86400")
        .body(Body::from(data))
        .unwrap())
}
```

```rust
use diesel::prelude::*;
use diesel::r2d2::{self, ConnectionManager};
use diesel_migrations::{EmbeddedMigrations, MigrationHarness, embed_migrations};

pub type LocalPool = r2d2::Pool<ConnectionManager<SqliteConnection>>;

const MIGRATIONS: EmbeddedMigrations = embed_migrations!("migrations");

pub fn create_local_pool() -> LocalPool {
    let data_dir = dirs::data_dir()
        .expect("Could not determine data directory")
        .join("dioxus_music");
    std::fs::create_dir_all(&data_dir).expect("Could not create data directory");

    let db_path = data_dir.join("cache.db");
    let database_url = db_path.to_str().expect("Invalid DB path").to_string();

    let manager = ConnectionManager::<SqliteConnection>::new(&database_url);
    let pool = r2d2::Pool::builder()
        .max_size(4)
        .build(manager)
        .expect("Could not create SQLite pool");

    // Run migrations
    let mut conn = pool.get().expect("Could not get connection");
    conn.run_pending_migrations(MIGRATIONS)
        .expect("Could not run migrations");

    // Enable WAL mode for better concurrent read/write
    diesel::sql_query("PRAGMA journal_mode=WAL;")
        .execute(&mut conn)
        .ok();

    pool
}
```

```rust
use dioxus::prelude::*;
use std::time::Duration;

#[derive(Debug, Clone, Copy, PartialEq)]
pub enum ConnectionStatus {
    Online,
    Offline,
    Unknown,
}

/// Provides a reactive ConnectionStatus signal that pings the server periodically.
pub fn use_connectivity(server_url: &str) -> Signal<ConnectionStatus> {
    let mut status = use_signal(|| ConnectionStatus::Unknown);
    let url = format!("{server_url}/api/health");

    use_resource(move || {
        let url = url.clone();
        async move {
            let client = reqwest::Client::new();
            loop {
                let result = client
                    .get(&url)
                    .timeout(Duration::from_secs(5))
                    .send()
                    .await;

                let new_status = match result {
                    Ok(resp) if resp.status().is_success() => ConnectionStatus::Online,
                    _ => ConnectionStatus::Offline,
                };

                if *status.peek() != new_status {
                    status.set(new_status);
                }

                tokio::time::sleep(Duration::from_secs(30)).await;
            }
        }
    });

    status
}
```

**Step 1: Implement sync engine**

Create `packages/desktop/src/sync_engine.rs`:
```rust
use crate::local_db::LocalPool;
use crate::local_models::*;
use crate::local_schema;
use diesel::prelude::*;
use dioxus_music_api::models::{PlaylistSummary, TrackSummary};
use tracing;
use uuid::Uuid;

pub struct SyncEngine {
    pool: LocalPool,
    server_url: String,
    client: reqwest::Client,
}

impl SyncEngine {
    pub fn new(pool: LocalPool, server_url: String) -> Self {
        Self {
            pool,
            server_url,
            client: reqwest::Client::new(),
        }
    }

    /// Run a full incremental sync: fetch changes since last_sync, upsert locally.
    pub async fn sync(&self) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let last_sync = self.get_last_sync()?;
        tracing::info!("Starting sync since {last_sync}");

        // Sync tracks
        let tracks: Vec<TrackSummary> = if last_sync == "1970-01-01T00:00:00Z" {
            // First sync: get everything
            dioxus_music_api::get_library().await?
        } else {
            let since = last_sync
                .parse::<chrono::DateTime<chrono::Utc>>()
                .unwrap_or_default();
            dioxus_music_api::get_library_since(since).await?
        };

        if !tracks.is_empty() {
            tracing::info!("Syncing {} tracks", tracks.len());
            self.upsert_tracks(&tracks)?;
        }

        // Sync playlists
        let playlists: Vec<PlaylistSummary> = if last_sync == "1970-01-01T00:00:00Z" {
            dioxus_music_api::get_playlists().await?
        } else {
            let since = last_sync
                .parse::<chrono::DateTime<chrono::Utc>>()
                .unwrap_or_default();
            dioxus_music_api::get_playlists_since(since).await?
        };

        if !playlists.is_empty() {
            tracing::info!("Syncing {} playlists", playlists.len());
            self.upsert_playlists(&playlists)?;
        }

        // Sync deletions
        if last_sync != "1970-01-01T00:00:00Z" {
            let since = last_sync
                .parse::<chrono::DateTime<chrono::Utc>>()
                .unwrap_or_default();
            let deleted_ids: Vec<Uuid> =
                dioxus_music_api::get_deleted_tracks_since(since).await?;
            if !deleted_ids.is_empty() {
                tracing::info!("Removing {} deleted tracks", deleted_ids.len());
                self.remove_tracks(&deleted_ids)?;
            }
        }

        // Sync playlist tracks for all playlists
        self.sync_all_playlist_tracks().await?;

        // Update last_sync
        let now = chrono::Utc::now().to_rfc3339();
        self.set_meta("last_sync", &now)?;

        tracing::info!("Sync complete");
        Ok(())
    }

    fn get_last_sync(&self) -> Result<String, Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        use local_schema::cache_meta::dsl::*;
        let result: Option<String> = cache_meta
            .filter(key.eq("last_sync"))
            .select(value)
            .first(&mut conn)
            .optional()?;
        Ok(result.unwrap_or_else(|| "1970-01-01T00:00:00Z".to_string()))
    }

    fn set_meta(
        &self,
        k: &str,
        v: &str,
    ) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        diesel::replace_into(local_schema::cache_meta::table)
            .values(&CacheMeta {
                key: k.to_string(),
                value: v.to_string(),
            })
            .execute(&mut conn)?;
        Ok(())
    }

    fn upsert_tracks(
        &self,
        tracks: &[TrackSummary],
    ) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        for track in tracks {
            let local = LocalTrack {
                id: track.id.to_string(),
                title: track.title.clone(),
                artist: track.artist.clone(),
                album: track.album.clone(),
                genre: track.genre.clone(),
                duration_secs: track.duration_secs,
                updated_at: track.updated_at.to_rfc3339(),
                local_path: None,       // Preserve existing if updating
                cached_at: None,         // Preserve existing if updating
                sync_status: "not_cached".to_string(),
            };
            // Use INSERT OR REPLACE but preserve local_path/cached_at/sync_status
            // if the track already exists locally
            let existing: Option<(Option<String>, Option<String>, String)> =
                local_schema::tracks::table
                    .filter(local_schema::tracks::id.eq(&local.id))
                    .select((
                        local_schema::tracks::local_path,
                        local_schema::tracks::cached_at,
                        local_schema::tracks::sync_status,
                    ))
                    .first(&mut conn)
                    .optional()?;

            if let Some((lp, ca, ss)) = existing {
                // Update metadata but preserve cache fields
                diesel::update(
                    local_schema::tracks::table.filter(local_schema::tracks::id.eq(&local.id)),
                )
                .set((
                    local_schema::tracks::title.eq(&local.title),
                    local_schema::tracks::artist.eq(&local.artist),
                    local_schema::tracks::album.eq(&local.album),
                    local_schema::tracks::genre.eq(&local.genre),
                    local_schema::tracks::duration_secs.eq(local.duration_secs),
                    local_schema::tracks::updated_at.eq(&local.updated_at),
                ))
                .execute(&mut conn)?;
            } else {
                diesel::insert_into(local_schema::tracks::table)
                    .values(&local)
                    .execute(&mut conn)?;
            }
        }
        Ok(())
    }

    fn upsert_playlists(
        &self,
        playlists: &[PlaylistSummary],
    ) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        for playlist in playlists {
            let existing_sync: Option<i32> = local_schema::playlists::table
                .filter(local_schema::playlists::id.eq(playlist.id.to_string()))
                .select(local_schema::playlists::sync_enabled)
                .first(&mut conn)
                .optional()?;

            let local = LocalPlaylist {
                id: playlist.id.to_string(),
                name: playlist.name.clone(),
                playlist_type: playlist.playlist_type.clone(),
                rules: None, // Fetched separately if needed
                updated_at: playlist.updated_at.to_rfc3339(),
                sync_enabled: existing_sync.unwrap_or(0),
            };

            diesel::replace_into(local_schema::playlists::table)
                .values(&local)
                .execute(&mut conn)?;
        }
        Ok(())
    }

    fn remove_tracks(
        &self,
        ids: &[Uuid],
    ) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        for id in ids {
            // TODO: also delete the cached audio file from disk
            diesel::delete(
                local_schema::tracks::table.filter(local_schema::tracks::id.eq(id.to_string())),
            )
            .execute(&mut conn)?;
        }
        Ok(())
    }

    async fn sync_all_playlist_tracks(
        &self,
    ) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;
        let playlist_ids: Vec<String> = local_schema::playlists::table
            .select(local_schema::playlists::id)
            .load(&mut conn)?;

        for pid_str in playlist_ids {
            let pid: Uuid = pid_str.parse()?;
            let tracks = dioxus_music_api::get_playlist_tracks(pid).await?;

            // Replace all playlist tracks
            diesel::delete(
                local_schema::playlist_tracks::table
                    .filter(local_schema::playlist_tracks::playlist_id.eq(&pid_str)),
            )
            .execute(&mut conn)?;

            for (i, track) in tracks.iter().enumerate() {
                let pt = LocalPlaylistTrack {
                    id: Uuid::new_v4().to_string(),
                    playlist_id: pid_str.clone(),
                    track_id: track.id.to_string(),
                    position: i as i32,
                };
                diesel::insert_into(local_schema::playlist_tracks::table)
                    .values(&pt)
                    .execute(&mut conn)?;
            }
        }
        Ok(())
    }
}
```

### Task 2.5: Build download manager

**Files:**
- Create: `packages/desktop/src/download_manager.rs`

This module handles:
1. Queuing tracks for download
2. Downloading audio files from `/stream/{id}`
3. Saving to local disk and updating SQLite
4. Opportunistic caching (save after stream playback)
5. LRU eviction for opportunistically cached tracks

**Step 1: Implement download manager**

Create `packages/desktop/src/download_manager.rs`:
```rust
use crate::local_db::LocalPool;
use crate::local_schema;
use diesel::prelude::*;
use std::path::PathBuf;
use tokio::sync::mpsc;
use tracing;
use uuid::Uuid;

pub struct DownloadManager {
    pool: LocalPool,
    server_url: String,
    client: reqwest::Client,
    cache_dir: PathBuf,
    tx: mpsc::UnboundedSender<DownloadRequest>,
}

struct DownloadRequest {
    track_id: Uuid,
    /// If true, this is an explicit sync download (never LRU-evicted).
    /// If false, it's an opportunistic cache (subject to eviction).
    is_sync: bool,
}

impl DownloadManager {
    pub fn new(pool: LocalPool, server_url: String) -> Self {
        let cache_dir = dirs::data_dir()
            .expect("Could not determine data directory")
            .join("dioxus_music")
            .join("audio_cache");
        std::fs::create_dir_all(&cache_dir).expect("Could not create cache directory");

        let (tx, rx) = mpsc::unbounded_channel();

        let manager = Self {
            pool: pool.clone(),
            server_url: server_url.clone(),
            client: reqwest::Client::new(),
            cache_dir: cache_dir.clone(),
            tx,
        };

        // Spawn background download worker
        tokio::spawn(Self::download_worker(
            rx,
            pool,
            server_url,
            cache_dir,
            reqwest::Client::new(),
        ));

        manager
    }

    /// Queue a track for download (sync or opportunistic cache).
    pub fn queue_download(&self, track_id: Uuid, is_sync: bool) {
        let _ = self.tx.send(DownloadRequest { track_id, is_sync });
    }

    /// Queue all tracks in synced playlists for download.
    pub fn sync_playlists(&self) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;

        // Get all playlist_ids where sync_enabled = 1
        let synced_playlist_ids: Vec<String> = local_schema::playlists::table
            .filter(local_schema::playlists::sync_enabled.eq(1))
            .select(local_schema::playlists::id)
            .load(&mut conn)?;

        // For each, get track IDs that are not yet cached
        for pid in synced_playlist_ids {
            let uncached_track_ids: Vec<String> = local_schema::playlist_tracks::table
                .inner_join(
                    local_schema::tracks::table
                        .on(local_schema::tracks::id.eq(local_schema::playlist_tracks::track_id)),
                )
                .filter(local_schema::playlist_tracks::playlist_id.eq(&pid))
                .filter(local_schema::tracks::sync_status.ne("cached"))
                .select(local_schema::tracks::id)
                .load(&mut conn)?;

            for tid_str in uncached_track_ids {
                if let Ok(tid) = tid_str.parse::<Uuid>() {
                    self.queue_download(tid, true);
                }
            }
        }

        Ok(())
    }

    async fn download_worker(
        mut rx: mpsc::UnboundedReceiver<DownloadRequest>,
        pool: LocalPool,
        server_url: String,
        cache_dir: PathBuf,
        client: reqwest::Client,
    ) {
        while let Some(req) = rx.recv().await {
            let url = format!("{}/stream/{}", server_url, req.track_id);
            tracing::info!("Downloading track {}", req.track_id);

            match client.get(&url).send().await {
                Ok(resp) if resp.status().is_success() => {
                    // Determine file extension from Content-Type
                    let ext = resp
                        .headers()
                        .get("content-type")
                        .and_then(|v| v.to_str().ok())
                        .map(|ct| match ct {
                            "audio/flac" => "flac",
                            "audio/mpeg" => "mp3",
                            "audio/ogg" => "ogg",
                            _ => "bin",
                        })
                        .unwrap_or("bin");

                    let filename = format!("{}.{}", req.track_id, ext);
                    let file_path = cache_dir.join(&filename);

                    match resp.bytes().await {
                        Ok(bytes) => {
                            if let Err(e) = tokio::fs::write(&file_path, &bytes).await {
                                tracing::error!("Failed to write file: {e}");
                                continue;
                            }

                            // Update local DB
                            let mut conn = match pool.get() {
                                Ok(c) => c,
                                Err(e) => {
                                    tracing::error!("DB pool error: {e}");
                                    continue;
                                }
                            };

                            let now = chrono::Utc::now().to_rfc3339();
                            let tid = req.track_id.to_string();
                            let path_str =
                                file_path.to_str().unwrap_or_default().to_string();

                            let _ = diesel::update(
                                local_schema::tracks::table
                                    .filter(local_schema::tracks::id.eq(&tid)),
                            )
                            .set((
                                local_schema::tracks::local_path.eq(&path_str),
                                local_schema::tracks::cached_at.eq(&now),
                                local_schema::tracks::sync_status.eq("cached"),
                            ))
                            .execute(&mut conn);

                            tracing::info!("Downloaded track {} -> {}", req.track_id, filename);
                        }
                        Err(e) => tracing::error!("Failed to read response: {e}"),
                    }
                }
                Ok(resp) => tracing::error!("Download failed: HTTP {}", resp.status()),
                Err(e) => tracing::error!("Download request failed: {e}"),
            }
        }
    }

    /// Evict opportunistically cached tracks (not in synced playlists) using LRU
    /// until total cache size is under the configured limit.
    pub fn evict_if_needed(&self) -> Result<(), Box<dyn std::error::Error + Send + Sync>> {
        let mut conn = self.pool.get()?;

        // Get cache limit
        let limit_str: String = local_schema::cache_meta::table
            .filter(local_schema::cache_meta::key.eq("cache_limit_bytes"))
            .select(local_schema::cache_meta::value)
            .first(&mut conn)?;
        let limit: u64 = limit_str.parse().unwrap_or(2_147_483_648);

        // Calculate current cache size
        let cached_paths: Vec<(String, Option<String>)> = local_schema::tracks::table
            .filter(local_schema::tracks::sync_status.eq("cached"))
            .select((local_schema::tracks::id, local_schema::tracks::local_path))
            .load(&mut conn)?;

        let mut total_size: u64 = 0;
        for (_, path) in &cached_paths {
            if let Some(p) = path {
                if let Ok(meta) = std::fs::metadata(p) {
                    total_size += meta.len();
                }
            }
        }

        if total_size <= limit {
            return Ok(());
        }

        // Get eviction candidates: cached tracks NOT in any synced playlist, ordered by cached_at ASC (oldest first)
        let synced_playlist_ids: Vec<String> = local_schema::playlists::table
            .filter(local_schema::playlists::sync_enabled.eq(1))
            .select(local_schema::playlists::id)
            .load(&mut conn)?;

        let synced_track_ids: Vec<String> = if synced_playlist_ids.is_empty() {
            vec![]
        } else {
            local_schema::playlist_tracks::table
                .filter(local_schema::playlist_tracks::playlist_id.eq_any(&synced_playlist_ids))
                .select(local_schema::playlist_tracks::track_id)
                .distinct()
                .load(&mut conn)?
        };

        // Get evictable tracks: cached but not in synced playlists
        let evictable: Vec<(String, Option<String>)> = local_schema::tracks::table
            .filter(local_schema::tracks::sync_status.eq("cached"))
            .filter(local_schema::tracks::id.ne_all(&synced_track_ids))
            .order(local_schema::tracks::cached_at.asc())
            .select((local_schema::tracks::id, local_schema::tracks::local_path))
            .load(&mut conn)?;

        for (tid, path) in evictable {
            if total_size <= limit {
                break;
            }

            if let Some(p) = &path {
                if let Ok(meta) = std::fs::metadata(p) {
                    total_size -= meta.len();
                }
                let _ = std::fs::remove_file(p);
            }

            diesel::update(local_schema::tracks::table.filter(local_schema::tracks::id.eq(&tid)))
                .set((
                    local_schema::tracks::local_path.eq::<Option<String>>(None),
                    local_schema::tracks::cached_at.eq::<Option<String>>(None),
                    local_schema::tracks::sync_status.eq("not_cached"),
                ))
                .execute(&mut conn)?;

            tracing::info!("Evicted track {tid} from cache");
        }

        Ok(())
    }
}
```

```kotlin
class MusicBrowserService : MediaBrowserServiceCompat() {
    private lateinit var session: MediaSessionCompat
    private lateinit var db: AppDatabase

    override fun onCreate() {
        super.onCreate()
        db = AppDatabase.getInstance(this)
        session = MusicMediaSession.create(this)
        sessionToken = session.sessionToken
    }

    override fun onGetRoot(
        clientPackageName: String,
        clientUid: Int,
        rootHints: Bundle?
    ): BrowserRoot {
        return BrowserRoot("root", null)
    }

    override fun onLoadChildren(
        parentId: String,
        result: Result<MutableList<MediaBrowserCompat.MediaItem>>
    ) {
        result.detach()

        when (parentId) {
            "root" -> {
                result.sendResult(mutableListOf(
                    browsableItem("library", "Library"),
                    browsableItem("playlists", "Playlists"),
                    browsableItem("downloaded", "Downloaded"),
                ))
            }
            "library" -> {
                // Load all tracks from Room, convert to MediaItems
                loadLibraryItems(result)
            }
            "playlists" -> {
                // Load playlist names from Room
                loadPlaylistItems(result)
            }
            "downloaded" -> {
                // Load only cached tracks
                loadDownloadedItems(result)
            }
            else -> {
                if (parentId.startsWith("playlist:")) {
                    val playlistId = parentId.removePrefix("playlist:")
                    loadPlaylistTracks(playlistId, result)
                } else {
                    result.sendResult(mutableListOf())
                }
            }
        }
    }

    // ... helper methods for building MediaItems from Room queries
}
```

### Dioxus 0.7 Mobile Interop Research

Task 3.4 (UniFFI bridge) requires researching how Dioxus 0.7 handles native platform interop on mobile. Check:
- `dioxus/packages/mobile/` in the Dioxus source
- Any platform channel or FFI mechanism in Dioxus 0.7 docs
- How `dx serve --platform android` packages the Rust binary into the APK

### Database Choice Rationale

- **Server (Postgres)**: Existing, battle-tested, async with diesel-async
- **Desktop (SQLite)**: Single-user, embedded, Diesel already supports it, no network dependency
- **Android (Room)**: Android's recommended persistence layer, integrates with Android Auto and lifecycle
- **iOS (Core Data)**: Apple's recommended persistence, integrates with CloudKit if ever needed

### Testing Strategy

No tests currently exist. As each phase is implemented, add:
- Unit tests for sync engine logic (mock server responses, verify SQLite state)
- Integration tests for download manager (mock HTTP, verify file caching)
- UI tests deferred until core logic is stable

### Order of Operations

Phases are designed to be sequential but some parallelism is possible:
- Phase 1 (server) must complete first
- Phase 2 (desktop) and Phase 3 (mobile UI) can overlap
- Phase 4 (Android Auto) requires Phase 3
- Phase 5 (iOS/CarPlay) can start alongside Phase 4
